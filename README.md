## 1. Prompt-Engineer-Framework
O Objetivo-central do Portfólio 'Prompt-Engineer-Framework' é de explorar como diferentes tipos de prompts fazem com que a IA responda de formas diferentes, fiz isso analisando a estrutura de dois Formatos de Prompt(Básico x Avançado). O Repositório do Projeto descreve por completo toda a análise que fiz de cada um dos dois prompts, aqui, exponho algumas observações feitas sobre os parâmetros e detalhes que o formato Avançado tinha e o Básico não.
